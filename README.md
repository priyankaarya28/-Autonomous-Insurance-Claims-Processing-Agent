# Autonomous Insurance Claims Processing Agent

This project is a lightweight, browser-based **Autonomous Insurance Claims Processing Agent** built using **HTML, CSS, and JavaScript**.

The agent processes FNOL (First Notice of Loss) documents, extracts key information, validates mandatory fields, classifies claims, and routes them to the appropriate workflow based on predefined business rules.
